# Welcome to mdshards

You're looking at a live, public **demo** of **mdshards** — a browser editor for
a vault of plain markdown notes, synced live between everyone connected.

> ### 🚧 Please read — this is a shared public demo
>
> - 📝 **This page is read-only.** You can type here, but your changes are **not
>   saved and not shared** with anyone else.
> - ⬆️ **Uploads are disabled.** `Cmd/Ctrl-U` opens the file picker, but nothing
>   is sent.
> - 💾 **Disk space is limited**, and the **whole vault is wiped every 2 hours** —
>   don't keep anything here you care about.
> - 🙏 **Be gentle.** Other people are poking at this same vault right now.

## Try it

Every note *except this one* is fully editable, and you can make your own:

| Shortcut | What it does |
| --- | --- |
| `Cmd/Ctrl-K` | Quick switcher — go to or create a note |
| `Cmd/Ctrl-Shift-K` | Rename the current note |
| `Cmd/Ctrl-Backspace` | Delete a note |
| `Cmd/Ctrl-E` | Emoji picker — inserts a `:shortcode:` |
| `Cmd/Ctrl-Alt-O` | Editor options (vim mode, line numbers) |
| `Cmd/Ctrl-U` | Upload — **disabled in this demo** |

## Markdown, rendered as you type

Text can be **bold**, *italic*, ~~struck through~~, or `inline code`. Internal
links use `[[wikilinks]]`; external links are shown but not clickable.
